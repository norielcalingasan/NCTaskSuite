NC TaskSuite (Client Readme)
Version: 1.0.9 (Beta)

What is NC TaskSuite?
- A task manager for solo work, private local teams, and optional shared workflows.

What is included in this folder?
- NC TaskSuite.exe (main app)
- NCUpdater.exe (optional updater)
- Features.txt (feature list)
- Changelogs.txt (version history)

How to run
1) Double-click NC TaskSuite.exe
2) If Windows SmartScreen appears, click More info > Run anyway.

Updates
- Run NCUpdater.exe when you want to update, or follow the in-app update prompt.

Data & privacy
- By default, tasks are stored locally on your PC.
- Auto-save and backups are stored under %APPDATA%\NC_TaskSuite.
- Cloud Workspace is optional and uses your own Firebase configuration.

Support
- For help or license support, contact your NC TaskSuite provider.
